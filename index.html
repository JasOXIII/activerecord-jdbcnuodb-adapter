Welcome to the JRuby ActiveRecord NuoDB JDBC Adapter Project

This site contains details on how to install and use the JRuby ActiveRecord NuoDB JDBC Adapter. To install the driver simply run the following command for pre-release builds:

$ gem install activerecord-jdbcnuodb-adapter --pre

For release builds it's simply:

$ gem install activerecord-jdbcnuodb-adapter
